# 블로그 디자인 방향 — "주장하는 종이"

> 이 문서는 재디자인의 **방향성(주장)** 을 고정하기 위한 기준선이다.
> 개별 구현이 흔들릴 때 여기로 돌아와 "이 결정이 주장과 맞는가"를 판단한다.

---

## 0. 한 줄 주장 (Thesis)

> **"이건 소비용 피드가 아니라, 정독할 인쇄물이다."**

밝은 종이 위에 잉크블루 단색으로, 논문처럼 조판한다.
디자인은 방문자를 즐겁게 하려 애쓰지 않고 **읽게 만든다.** 절제가 곧 권위다.

이 주장은 콘텐츠의 정체성("정의부터 증명까지 · 정답보다 과정")과 한 몸이며,
개발자 블로그 바다에서 **가장 희소한 포지션**을 의도적으로 택한 것이다.
(대부분이 다크+네온·터미널 미감으로 몰려 오히려 흔하다.)

---

## 1. 확정된 정체성 (요약)

| 축 | 결정 |
|---|---|
| **바탕(Mode)** | 밝은 종이 우선. 다크는 토글로 유지하되 **라이트가 정본, 다크는 번역본** |
| **활자(Type)** | 하이브리드 **3음성** — 제목·UI(산세) / 본문(세리프) / 코드·수식(모노) |
| **강조색(Accent)** | **잉크블루 `#1C3D5C` 단 하나.** 적색은 "주의"에만 허용하는 예외 |
| **레이아웃(Layout)** | **조용한 레일(Quiet Rail)** — 사이드바는 항상 있되 접힌 레일로 물러나고, 본문은 중앙 측정폭 고정 |
| **모션(Motion)** | 정적 기반 + **기능적 예외 3종만**(읽기 진행 바 · TOC 활성 · 레일 펼침) |
| **논문 장치(Apparatus)** | **정식 정리 환경** — 정의/정리/증명(∎) 번호 체계, 섹션·그림 번호, 각주 |
| **홈(Home)** | **에디토리얼(따뜻한 저널 표지)** — 강령 헤드라인 + 추천 글 + 피드 |
| **회고(Retrospective)** | 블로그를 따로 파지 않는다. **같은 지붕, 두 번째 조판(음성)** |

## 2. 결정 상세 + 근거

### 2.1 바탕 — 밝은 종이 우선
- **결정**: 크림/오프화이트 바탕 + 진한 잉크. 다크 토글 유지, 위계는 라이트 > 다크.
- **왜**: 세리프 본문과 종이 바탕은 한 몸. "정독할 인쇄물"이라는 주장을 바탕색만으로 관철한다.
- **함의**: 기존 오렌지 glow·radial 그라디언트·네온 계열은 **거짓말이 되므로 폐기**한다. 종이는 빛나지 않는다.

### 2.2 활자 — 하이브리드 3음성
- **제목·UI**: `Pretendard` (기존 유지) — 기하학적 산세, 세리프 본문과 대비.
- **본문**: `Noto Serif KR`(한글) + `Newsreader`(라틴) — "이 텍스트는 정독 대상"이라는 선언. Newsreader의 이탤릭을 수식 변수·강조·"증명." 도입어에 활용.
- **코드·수식**: `JetBrains Mono`(기존 유지) + KaTeX(Computer Modern) — 기계적으로 유지해 본문과 대비.
- **비고**: 실물 렌더 확인 후 조정 가능(한글을 나눔명조로, 또는 라틴 생략 등).

### 2.3 강조색 — 잉크블루 단색
- **결정**: `#1C3D5C` 하나. 만년필 잉크 = "손으로 쓴 수학 노트"의 서사.
- **규율**: 색을 늘리면 즉시 "블로그 템플릿"으로 추락한다. 위계는 색이 아니라 **라벨·괘선·번호·농담(shade)** 으로 만든다.
- **예외**: 적색 `#9C4436` 은 오직 "주의(warn)"의 의미를 전할 때만. 브랜드 강조색이 아니다.

### 2.4 레이아웃 — 조용한 레일
- **결정**: 사이드바(탐색)는 항상 존재하나 평소엔 아이콘 레일로 접혀 물러난다. 호버/고정으로 본문 **위에** 펼쳐지며 본문을 밀지 않는다. 본문은 사이드바 상태와 무관하게 **중앙 측정폭(~40em) 고정**.
- **왜**: C(사이드바)의 탐색 접근성 + A(중앙 단)의 독서 몰입을 모두 취한다.
- **운영**: **글 페이지 = 레일 접힘 기본**(몰입) / **홈·목록 = 레일 펼침 기본**(색인).

### 2.5 모션 — 정적 기반 + 기능적 예외 3
- **철학**: 종이는 튀지 않는다. hover는 이동·glow가 아니라 **잉크의 변화**(색·밑줄)로만.
- **허용**: ① 읽기 진행 바 ② TOC 활성 위치 ③ 레일 펼침/접힘.
- **폐기**: stagger fadeInUp, translateX hover, box-shadow glow, 바 차트 애니메이션.
- `prefers-reduced-motion`에서 위 3종 전부 즉시 정지.

### 2.6 논문 장치 — 정식 정리 환경
- **환경**: `정의 / 정리 / 보조정리 / 따름정리 / 증명(∎)` 을 **번호 매겨** 조판(예: 정의 2.1, 정리 2.2).
- **구조**: 섹션 번호(1, 1.1…), 번호 붙은 그림(그림 2.1), 각주.
- **콜아웃 3종으로 통합**(기존 5색 폐기):
  - `요점` — 잉크블루 괘선 (기존 key·tip 통합)
  - `註` — 회색 괘선 (기존 info·simple 통합)
  - `주의` — 적색 괘선 (유일한 세만틱 예외색)
- **난이도**: 5색 배지 → **점 표기 ●●●○○**(잉크 단색).

### 2.7 홈 — 에디토리얼(따뜻한 저널 표지)
- **결정**: 강령 헤드라인 + 추천 글 카드 + 최근 피드 + 주제별 편수. 종이·잉크블루로 재단장.
- **왜**: 순수 색인(도서관 표제지)은 정합적이나 방문자에게 **너무 차갑다.** 첫 화면은 사람 냄새와 유입 동선을 남긴다.
- **원칙**: 읽기 화면은 엄격한 논문, **첫 화면은 따뜻하게.** 이 온도차를 의도적으로 유지한다.

### 2.8 회고 — 같은 지붕, 두 번째 조판
- **결정**: 회고용 블로그를 분리하지 않는다. 카테고리(`회고`/`노트`)로 두고 **시각적 register만** 다르게 — 정식 글이 "논문 조판"이면 회고는 "저자의 육필 노트"(더 좁고, 더 개인적이고, 장식을 덜어낸).
- **왜**: 주장은 통일된 정체성에서 나온다. 쪼개면 밀도가 절반이 되고 SEO·관리도 분산된다. 오히려 두 층위(정식/노트)가 엄밀함에 인간미를 더한다.

## 3. 색 토큰 (초안)

```
--paper        #FAF9F6   /* 종이 바탕 (다크 팔레트는 별도 확정) */
--ink          #1B1917   /* 본문 잉크 (따뜻한 near-black) */
--muted        #6B6862   /* 보조 텍스트 */
--hair         #E5E2DA   /* 괘선·경계 */
--accent       #1C3D5C   /* 잉크블루 — 유일한 강조 */
--accent-tint  color-mix(accent 7%, paper)   /* 정리 박스 배경 */
--warn         #9C4436   /* '주의'에만 쓰는 예외 세만틱 적색 */
```
- 난이도: `--accent` 농담/점 표기로 표현(별도 색상 없음).
- **다크 모드 팔레트는 미확정** — 라이트 확정 후 별도로 정한다.
