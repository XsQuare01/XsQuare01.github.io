# Footer 구현 플랜

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 메인 콘텐츠 영역 하단에 이름·소개 + copyright·GitHub·이메일 링크로 구성된 2단 footer를 추가한다.

**Architecture:** `BaseLayout.astro`의 `.main-wrap` 내부 `<slot />` 아래에 `<footer>` 마크업을 삽입하고, `global.css`에 스타일을 추가한다. `PostLayout.astro`는 `BaseLayout`을 상속하므로 별도 수정 불필요.

**Tech Stack:** Astro, CSS (CSS Variables)

---

## 파일 변경 목록

| 파일 | 작업 |
|------|------|
| `src/layouts/BaseLayout.astro` | `<slot />` 아래에 `<footer class="site-footer">` 마크업 추가 |
| `src/styles/global.css` | `.site-footer` 관련 스타일 4개 클래스 추가 |

---

### Task 1: CSS 스타일 추가

**Files:**
- Modify: `src/styles/global.css`

- [ ] **Step 1: `global.css` 끝에 footer 스타일 추가**

`src/styles/global.css` 파일의 맨 아래(Mobile 미디어 쿼리 위)에 다음을 추가한다:

```css
/* ── Site Footer ── */
.site-footer {
  border-top: 1px solid var(--border);
  padding: 2.5rem 0;
  margin-top: 3rem;
}

.site-footer-top {
  display: flex;
  flex-direction: row;
  align-items: center;
  gap: 0.5rem;
  margin-bottom: 0.5rem;
}

.site-footer-name {
  font-weight: 700;
  color: var(--text);
}

.site-footer-bio {
  color: var(--text-muted);
  font-size: 0.85rem;
}

.site-footer-bottom {
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
  align-items: center;
  gap: 0.5rem 1rem;
  font-size: 0.8rem;
  color: var(--text-dim);
  margin-top: 0.5rem;
}
```

정확한 삽입 위치: 파일의 `/* Mobile */` 블록 바로 위.

- [ ] **Step 2: 커밋**

```bash
git add src/styles/global.css
git commit -m "style: site-footer 스타일 추가"
```

### Task 2: BaseLayout에 footer 마크업 추가

**Files:**
- Modify: `src/layouts/BaseLayout.astro`

- [ ] **Step 1: `<slot />` 아래에 footer 마크업 삽입**

`BaseLayout.astro`에서 아래 부분을 찾는다:

```astro
  <div class="main-wrap">
    <slot />
  </div>
```

다음으로 교체한다:

```astro
  <div class="main-wrap">
    <slot />
    <footer class="site-footer">
      <div class="site-footer-top">
        <span class="site-footer-name">XsQuare01</span>
        <span class="site-footer-bio">기록하는 개발자</span>
      </div>
      <div class="site-footer-bottom">
        <span>© {new Date().getFullYear()} XsQuare01</span>
        <a href="https://github.com/XsQuare01" target="_blank" rel="noopener noreferrer">GitHub</a>
        <a href="mailto:mystic6113@naver.com">이메일</a>
      </div>
    </footer>
  </div>
```

- [ ] **Step 2: 로컬에서 동작 확인**

```bash
npm run dev
```

브라우저에서 `http://localhost:4321` 열어 다음 확인:
- 메인 페이지(`/`) 하단에 footer가 보이는가
- 포스트 페이지(`/blog/...`) 하단에도 footer가 보이는가
- GitHub 링크 클릭 시 새 탭으로 열리는가
- 이메일 링크 클릭 시 메일 클라이언트가 열리는가
- 모바일 뷰(`max-width: 768px`)에서 footer가 페이지 맨 아래에 위치하는가

- [ ] **Step 3: 커밋**

```bash
git add src/layouts/BaseLayout.astro
git commit -m "feat: 메인 콘텐츠 하단에 site-footer 추가"
```
