## 주요 시리즈

### 계산 이론

오토마타, 튜링 머신, 복잡도 클래스의 수학적 기반

| 글 | 요약 |
|---|---|
| [Interesting Number Paradox](https://xsquare01.github.io/blog/interesting-number-paradox) | 수학적 정의의 엄밀함이 왜 중요한지 |
| [Problem & Solution](https://xsquare01.github.io/blog/problem-and-solution) | Decision Problem, Language, 튜링 머신 |
| [DFA](https://xsquare01.github.io/blog/dfa) | 결정론적 유한 오토마타와 Pumping Lemma |
| [NFA](https://xsquare01.github.io/blog/nfa) | 비결정론적 유한 오토마타와 Powerset Construction |
| [DPDA / NPDA](https://xsquare01.github.io/blog/dpda-npda) | 스택을 가진 오토마타 |
| [Turing Machine](https://xsquare01.github.io/blog/turing-machine) | 계산의 극한, Universal TM |
| [Classes](https://xsquare01.github.io/blog/classes) | 계산 가능성 클래스 D, E, co-E |
| [집합의 크기 (Cardinality)](https://xsquare01.github.io/blog/cardinality) | 칸토어의 대각선 논법 |

| [CRT](https://xsquare01.github.io/blog/crt) | 중국인의 나머지 정리와 RSA 최적화 |
| [소수 판별 (Find Prime)](https://xsquare01.github.io/blog/find-prime) | 밀러-라빈 확률적 소수 판별법 |

| [Before Physics](https://xsquare01.github.io/blog/before-physics) | 양자 컴퓨터 이전에 알아야 할 것들 |
